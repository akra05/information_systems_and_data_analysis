# Information Systems and Data Analysis – TU Berlin

This repository contains all assignments from the course **Information Systems and Data Analysis** at TU Berlin.

## Assignments

| Folder | Topic |
|---|---|
| `assignment_1/` | **Database Schema Design** – Designed a relational database schema from scratch based on an EER diagram, including complex integrity constraints, and extended an existing incomplete database |
| `assignment_2/` | **SQL Queries** – Wrote queries ranging from simple selections to complex aggregations and joins |
| `assignment_3/` | **Data Stream Processing** – Handled and analyzed continuous data streams |

## Technologies
 
- **DuckDB** – relational in-process database
- **BigQuery** – SQL queries on large datasets
- **Python** – data stream processing
- SQL (DDL & DML) – schema design, CTEs, set operations, aggregations
- Complex data types: `STRUCT`, `VARCHAR[]`, `UUID`
- Count-Min Sketch – probabilistic frequency estimation on streams
- Sliding & landmark time windows
